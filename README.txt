# SmartSurveillance

**SmartSurveillance** is an AI-powered security system designed to perform **real-time weapon, pose, and object detection** using YOLOv5 and YOLOv8.  
It is optimized for surveillance environments, smart cameras, and research in intelligent monitoring.
